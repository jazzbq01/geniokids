# Mapeo pedagógico Matemática 1.º primaria - 250 actividades

Referencia: Cuaderno de trabajo Matemática 1, primer grado 2022.

El contenido no copia páginas ni imágenes del PDF. Usa el índice y los temas como referencia pedagógica para crear actividades originales en GenioKids.

## Distribución

| Nivel | Enfoque | Actividades |
|---|---|---:|
| Semilla Genial | Conteo, números 1 al 10, juntar y quitar cantidades pequeñas | 50 |
| Explorador Curioso | Comparación, orden, ordinales, patrones, tablas y figuras planas | 50 |
| Aventurero Ágil | Decenas, números hasta 20, representación, juntar, agregar y quitar | 50 |
| Estrella Brillante | Sumas, restas, gráficos, medición, recta numérica y geometría | 50 |
| Maestro Genio | Números hasta 50, dinero, doble, mitad, igualdad y lógica | 50 |

## Regla de avance

El siguiente nivel se desbloquea si el estudiante:

- completa el nivel anterior, o
- obtiene promedio mínimo 15/20 en el nivel anterior.

## Carga futura de libros

Mientras los próximos libros usen los tipos actuales de actividad (`quiz`, `matching`, `memory`, `trace`, `story`, `drag-drop`) y la misma estructura de tablas, se agregan solo con nuevos archivos SQL de carga. No se toca el código.
